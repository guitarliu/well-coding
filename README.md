# well-coding
Project during first Job

## 内容

* 5个数据处理软件
    *  金属磨粒及颗粒度的反推算
    *  粘度、破乳化、酸值计算
    *  酸值-体积计算
    *  颗粒度均值计算与级数的确定
    *  色谱实验数据验证
## 注意事项

* 使用pyinstaller编译并打包程序为.exe格式, 源代码为python2.x版
* 上传的各个程序均为64位，无法在32位系统上使用
* 程序的使用要结合电脑位数，32位只能打包成32的.exe文件
